# 基于Rust actix_web以及maud的网页留言板
## 功能
1.以用户名+文本+图片/视频的方式进行留言

2.使用SQLite数据库长期存储

3.支持通过Web浏览器访问留言板

4.可通过Edge/Chrome浏览器安装PWA应用至桌面

## 使用
在[Release](https://github.com/sankuchuari/Web_Message_Board/releases/tag/v1.1)下载zip安装包解压后运行可执行文件即可启用留言板Web
